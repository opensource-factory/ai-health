# AI Health App

Full-stack AI-powered health tracking application.

## Stack

Frontend
- Next.js

Backend
- FastAPI

## Project Structure


ai-health
│
├── frontend
├── backend
└── .gitignore


## Running the Project

### Backend


cd backend
venv\Scripts\activate
uvicorn app.main:app --reload


### Frontend


cd frontend
npm run dev


Frontend: http://localhost:3000  
Backend: http://127.0.0.1:8000
